# Tennessee State Roleplay - Staff Management Bot

A Discord bot for managing TSRP staff operations.

## Setup

1. Get your bot token from Discord Developer Portal
2. Replace `YOUR_BOT_TOKEN_HERE` in `main.py`
3. Install dependencies: `pip install -r requirements.txt`
4. Run: `python main.py`
5. Use `/tsrp_dashboard` in Discord

## Features

- Create infractions with reasons
- Create promotions with reasons
- Manage custom embeds
- Configure staff ranks
- Configure infraction types
